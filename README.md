# 1. Créer et lancer l'installeur
```
nano install-cka-training.sh
chmod +x install-cka-training.sh
./install-cka-training.sh
```

# 2. Configurer les alias
```
cd cka-training
./setup-aliases.sh
source ~/.bashrc
```
# 3. Vérifier et démarrer
```
./test-setup.sh
./cka-training.sh
```
